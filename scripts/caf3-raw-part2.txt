Topic: Core Components & The OSI Model
Q1. An accountant at Circle Textiles uses a specialized ERP software to input daily financial transactions. In the context of IT systems architecture, this ERP software is classified as: A) System Software B) Application Software C) Middleware D) Hardware Interface Correct Answer: B Explanation: Application software consists of programs designed to help users perform specific business tasks (like accounting, word processing, or CRM), whereas system software (like an Operating System) manages the hardware
.
Q2. When a user clicks "Request Ride" on a ride-hailing app, the app formats this request into a message to be sent over the network. Which layer of the OSI model is directly interacting with this end-user application? A) Presentation Layer (Layer 6) B) Application Layer (Layer 7) C) Network Layer (Layer 3) D) Transport Layer (Layer 4) Correct Answer: B Explanation: The Application Layer (Layer 7) is the closest layer to the end-user. It provides services directly to the user's application, such as file transfers, email, or in-app requests
.
Q3. A healthcare portal encrypts patient data and compresses it before sending it to another hospital's database, ensuring the receiving system can correctly read the format. Which layer of the OSI model handles this translation, encryption, and compression? A) Presentation Layer (Layer 6) B) Session Layer (Layer 5) C) Data Link Layer (Layer 2) D) Physical Layer (Layer 1) Correct Answer: A Explanation: The Presentation Layer (Layer 6) acts as the data translator. It handles data formatting (e.g., JPEG, XML), encryption, and compression to ensure data from one system is readable by another
.
Q4. A telecommunications router evaluates multiple paths and decides the fastest, most efficient physical route to send data packets from Karachi to London. This routing function is performed at the: A) Transport Layer B) Network Layer C) Data Link Layer D) Physical Layer Correct Answer: B Explanation: The Network Layer determines the best physical path for the data to take across multiple networks and handles the routing of data packets
.
Q5. What is the primary role of the Operating System within IT architecture? A) To act as an intermediary between the hardware components and the application software. B) To design the visual graphical user interface (GUI) of websites. C) To exclusively manage the cloud-native containerization processes. D) To permanently store databases. Correct Answer: A Explanation: The operating system (e.g., Windows, Linux) acts as the crucial intermediary layer between the physical hardware and the application software, managing memory, storage, and processing power
.

This strategy is an example of: A) Vertical Scaling B) Horizontal Scaling C) Downscaling D) Middleware integration Correct Answer: B Explanation: Horizontal scaling (or scaling out) involves adding more machines or servers to a network to distribute the workload, which is highly common in cloud environments during traffic surges

This is an example of: A) Horizontal Scaling B) Cloud-Native Architecture C) Vertical Scaling D) Edge Computing Correct Answer: C Explanation: Vertical scaling (or scaling up) involves upgrading the existing hardware infrastructure of a single machine (e.g., adding more RAM or a faster CPU) to handle increased workloads
.
Q8. What is the primary limitation of choosing Vertical Scaling over Horizontal Scaling? A) It requires highly complex network configurations. B) It is limited by the maximum hardware capacity of a single physical machine. C) It cannot be used for monolithic applications. D) It drastically increases the number of servers to manage. Correct Answer: B Explanation: The main limitation of vertical scaling is that it is eventually capped by the physical hardware limits of a single machine (e.g., a motherboard can only hold so much RAM), whereas horizontal scaling can theoretically add endless machines
.

Q1. What is the primary feature of blockchain technology that ensures data cannot be altered or tampered with once it is recorded? Q2. In a blockchain network, what happens if a hacker attempts to alter a transaction within a specific block? A) The network immediately centralizes to protect the data. D) The transaction is approved but flagged for manual review. Correct Answer: B Explanation: If someone alters a transaction, the hash of that block changes. Because each block relies on the previous block’s hash, this disruption breaks the link and invalidates the entire subsequent chain
.
Q3. Which type of participant in a blockchain network stores a complete copy of the entire ledger and actively participates in validating and verifying transactions? .
Q4. What is the specific purpose of "hashing" in a blockchain environment? A) To increase transaction processing speed B) To create a unique digital fingerprint of the data within a block C) To replace the need for smart contracts D) To reduce the number of nodes required in the network Correct Answer: B Explanation: Hashing is used to create a unique digital fingerprint for the data in a block, linking it securely to the previous block in the chain
